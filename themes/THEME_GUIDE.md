# テーマ自作ガイド — どんな世界観でも作れるレシピ

テーマCSS = **トークン上書き + 少量の装飾**。レイアウトには一切手を触れない。
新しいテーマは以下の手順で30分で作れる。

## 手順

### 1. 世界観を3語で決める
例:「高級・静寂・和」「元気・こども・原色」「未来・闇・ネオン」

### 2. トークンを埋める（必須14個）

```css
.slide {
  /* 配色: 必ずコントラスト比を確認（本文 ink は bg に対して 7:1 以上） */
  --bg:       ;   /* 地色 */
  --bg-soft:  ;   /* カード地（bg と僅差で上品に） */
  --ink:      ;   /* 主文字色 */
  --ink-2:    ;   /* 副文字色 */
  --muted:    ;   /* 補足・番号 */
  --line:     ;   /* 罫線 */
  --accent:   ;   /* 主役色: バッジ・強調・kicker */
  --accent-2: ;   /* 二番手（罫・下線・差し色） */
  --on-accent:;   /* accent 上の文字色 */

  /* カバー面（表紙・章扉・締めの3枚が使う） */
  --cover-bg:    ;   /* 濃色にすると引き締まり、明色にすると軽やか */
  --cover-ink:   ;
  --cover-muted: ;

  /* フォント: 世界観の70%はここで決まる */
  --font-display: ;  /* 見出し用 */
  --font-body:    ;  /* 本文用 */
}
```

### 3. 形状トークンで「硬さ」を調整

| 世界観 | --radius | --card-border | --card-shadow |
|--------|---------|---------------|----------------|
| 誠実・企業 | 14-18px | 1px solid var(--line) | 薄いぼかし影 |
| ポップ・楽しい | 24-32px | 3px solid var(--ink) | 8px 8px 0 の単色ずらし影 |
| 高級・静寂 | 0-8px | 1px solid（金や墨） | なし |
| 未来・テック | 8-14px | 1px solid（発光色30%） | 0 0 40px の発光 |

### 4. フォントの選び方（Google Fonts + ローカルフォールバック）

| 世界観 | display フォント例 |
|--------|-------------------|
| ビジネス | "Noto Sans JP" / Hiragino Kaku Gothic ProN |
| ポップ | "M PLUS Rounded 1c" / Hiragino Maru Gothic ProN |
| 高級・和 | "Shippori Mincho" "Zen Old Mincho" / Hiragino Mincho ProN |
| フェミニン | "Zen Maru Gothic" |
| テック | "Zen Kaku Gothic New" + 英数 "Space Grotesk" |
| レトロ | "Zen Kaku Gothic Antique" "DotGothic16"（見出しのみ） |

`@import url('https://fonts.googleapis.com/css2?family=...&display=swap');` を先頭に。
**必ず日本語対応フォントを選ぶ**（英語専用フォントを本文に使うと日本語がフォールバックでちぐはぐになる。英語専用フォントは .eyebrow/.kicker 等の英字パーツだけに使うのは可）。

### 5. 装飾（deco）を2〜3個だけ足す

表紙と締めに置く控えめな装飾。**必ず `.deco` クラス配下に**（QC対象外にするため）。
例: グラデーション円 / 紙吹雪 / 罫線グリッド / ノイズ角丸 / 和柄の縁。
装飾は「多いほど安っぽい」。3個まで。

### 6. 微調整フック（任意）

- `.head h2 em` — 見出し強調語の表現（色変え / マーカー / 下線）
- `.card:nth-child(3n+1) .badge` … — バッジ色のローテーション
- `.section-num` — 章番号の演出（透かし / 縁取り）
- `.panel-head` / `.vs` — 比較の配色
- `.cover-title` — text-shadow などの飾り

## 品質チェックリスト（テーマ完成の定義）

- [ ] サンプルデッキをレンダリングして QC違反ゼロ
- [ ] 表紙・本文・締めを PNG で目視し「そのジャンルのプロの資料」に見える
- [ ] 本文文字色のコントラスト比 7:1 以上（補足文字も 4.5:1 以上）
- [ ] アクセント色の使用面積が全体の 2割以下（多いと安っぽい）
- [ ] 装飾が文字に重なっていない
